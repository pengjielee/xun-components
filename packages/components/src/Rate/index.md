## 参数

| 属性      | 说明         | 类型               | 默认值  |
| --------- | ------------ | ------------------ | ------- |
| value     | 值           | `string`           | -       |
| disabled  | 是否禁用     | `boolean`          | `false` |
| allowHalf | 是否允许半星 | `boolean`          | `false` |
| count     | 星的个数     | `number`           | `5`     |
| onChange  | 値改变的方法 | `(value) => void ` | -       |

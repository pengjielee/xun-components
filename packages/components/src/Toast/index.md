## 参数

| 参数     | 说明       | 类型      | 默认值                                               |
| -------- | ---------- | --------- | ---------------------------------------------------- |
| type     | 类型       | `string`  | 可选：`'success','failure','info','warning','error'` |
| content  | 内容       | `string`  | ``                                                   |
| hasMask  | 显示遮罩层 | `boolean` | `false`                                              |
| duration | 显示时间   | `number`  | `3000`                                               |

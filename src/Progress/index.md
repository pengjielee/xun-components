## 参数

| 参数        | 说明                | 类型     | 默认值    |
| ----------- | ------------------- | -------- | --------- |
| percent     | 百分比              | `number` | `0`       |
| strokeColor | 进度条颜色          | `string` | `#00bc70` |
| strokeWidth | 线条宽度，单位 `px` | `number` | `5`       |
